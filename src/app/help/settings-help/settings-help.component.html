Settings help

Maximum hours per day

This setting currently has no impact.

Working days

Select the weekdays from Sunday to Saturday, that you are supposed to work on. This setting affects the calculation of hours per day as well as the overtime. Absences during weekdays not configured as working days do not have any impact on the final accounting of the month.

Hours per day

This value is calculated using the settings for 'hours per week', 'pensum' and the number of selected 'working days'. Every full absence counts as the equivalent number of hours, that is displayed in this setting.

Current overtime

This number specifies all hours of overtime, that have been accumulated during all month, regardless wether they are already deleted or not. Whenever a month is deleted, the calculated month's overhours can still be saved automatically.
