Tasks help

Favorite tasks

For every selection list, that offers all active tasks, the favorite task will always be preselected.

Active tasks

Only active tasks will be displayed in selection lists. A task is active, if it is marked as active.

Delete a task

By pressing this button you can delete a task. Only tasks, that are not accounted in any month, can be deleted. In order to delete an accounted task, you need to delete the affected months first.
