Projects help

Favorite projects

For every selection list, that offers all active projects, the favorite project will always be preselected.

Active projects

Only active projects will be displayed in selection lists. A project is active, if it is marked as active and contains active tasks.

Delete a project

By pressing this button you can delete a project. Only projects, that are not accounted in any month, can be deleted. In order to delete an accounted project, you need to delete the affected months first.
